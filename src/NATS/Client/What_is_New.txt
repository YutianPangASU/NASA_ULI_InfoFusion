National Airspace Trajectory-Prediction System(NATS)

Version: beta 1.3

Changes in this release:

1. Fixed bug of infinite value of ROCD if given high max flight level

2. Provide detail simulation in GO AROUND and all HOLD_PATTERN flight phases

3. Add simulation handling the scenarios when aircrafts run out of runways in departing and landing

4. Fixed Gate-To-Gate Flight plan generator to parse the approach procedures that earlier had an issue.

5. Added Scilab and GNU Octave support to use NATS functionalities.
